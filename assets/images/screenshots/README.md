## Especificações das Imagens

- **Formato**: PNG (recomendado), JPG ou WebP
- **Dimensões**: Recomendado 1920x1200px (proporção 16:10)
- **Tamanho**: Máximo 500KB por imagem (otimize para web)
- **Qualidade**: Alta qualidade, mas otimizada para web

## Como Adicionar Screenshots

1. Capture screenshots reais da aplicação ou crie mockups profissionais
2. Otimize as imagens usando ferramentas como:
   - [TinyPNG](https://tinypng.com/)
   - [Squoosh](https://squoosh.app/)
   - [ImageOptim](https://imageoptim.com/)
3. Nomeie os ficheiros conforme a lista acima
4. Coloque os ficheiros nesta pasta (`assets/images/screenshots/`)
5. As imagens serão automaticamente carregadas na homepage

## Dicas para Screenshots Eficazes

- Use dados realistas mas genéricos (sem informações pessoais)
- Capture em diferentes tamanhos de ecrã (desktop, tablet)
- Mostre funcionalidades principais e interfaces mais atrativas
- Mantenha consistência visual entre as screenshots
- Considere adicionar anotações ou highlights em funcionalidades-chave

## Fallback

Se uma imagem não existir, será exibido um placeholder com fundo gradiente.
